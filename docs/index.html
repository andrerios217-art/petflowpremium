Pet Flow Premium

Esta é uma página temporária de demonstração publicada via GitHub Pages para testes rápidos de acesso e apresentação.

Sobre esta publicação

O sistema completo do Pet Flow Premium depende de backend FastAPI, banco PostgreSQL e execução com Docker. Por isso, o GitHub Pages consegue publicar apenas uma versão estática de apresentação.

O que funciona aqui

Página institucional temporária, links, textos, layout e apresentação visual.

O que não funciona no GitHub Pages

Login, agenda, linha de produção, PDV, APIs, banco de dados e qualquer recurso que dependa do backend.

Resumo técnico

Stack atual FastAPI, PostgreSQL, autenticação, módulos operacionais e Docker.
Uso ideal do GitHub Pages Landing page, demonstração visual, documentação ou vitrine temporária.
Uso ideal para sistema real Render, Railway, VPS ou outra hospedagem com suporte a backend e banco.
Objetivo desta página Facilitar teste de acesso público sem mexer no funcionamento do backend principal.
